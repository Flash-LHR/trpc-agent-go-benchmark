# Create Economic Snapshot for Three Countries

## When to use

Use this skill when you need to gather and compile economic data for three specific countries.

## Steps

1. Define the three countries to analyze along with their required codes and regions.
2. Use three API endpoints for each country: one for GDP data, one for economic indicators, and one for population data.
3. Collect necessary details such as GDP, GDP per capita, population, inflation rate, and unemployment rate.
4. Determine the economic power rank and development tier for each country based on the gathered data.
5. Format the collected data into a structured JSON output.

## Pitfalls

- Ensure correct country codes and region designations are used for API calls.
- Verify that all economic indicators are gathered as specified in the task.
- Validate the output structure matches the required JSON format before finalizing.
